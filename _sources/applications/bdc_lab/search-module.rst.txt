Search
======

.. image:: ../../img/dc-explorer/search-1.png
    :alt: Search Component
    :align: center

Search Features
+++++++++++++++

-   Choose from a set of resources to get data, between Data Cubes, Collections and Classifications (groups on the left side of the screen);
-   Choose from a set of **collection** to get data (check boxes inside the resources);
-   Filter the selected collections using the **location**. You can do this by drawing on the map with the select region button or by entering a location in the search field on the right tab of the location component;
-   Filter the selected collections using the **time range** by typing the date or selecting the date from the calendar;

Additional Features
+++++++++++++++++++

-   You can see the **full name** of a collection by hovering mouse over the collection name;
-   When selecting a collection, clicking on the switcher, the **time range** fields will be filled (You can change it);
-   Select Date by typing the date or selecting the date from the calendar;
